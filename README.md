## 使い方

`Markdownファイル`の中に`.md`ファイルを保存し、
GitPitchで見せたいスライド毎にフォルダ/ブランチ分けして
以下のURLでアクセス。

[https://gitpitch.com/yuki476/myslide/<ブランチ>?p=<フォルダ名>]
